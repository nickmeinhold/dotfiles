# dotfiles
Dotfiles for macOS.

### Updating

#### node & npm

Download installer from https://nodejs.org/en/download/

#### firebase

Run `curl -sL firebase.tools | upgrade=true bash`

Reference: https://firebase.google.com/docs/cli#macos
